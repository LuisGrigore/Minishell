t_environment * env_create ( void  )

Add or update an environment variable.

If the variable does not exist, it is created and added to the list. If the variable already exists, its value is updated.

Parameters
envEnvironment variable list.
nameName of the variable (must not be NULL).
valueValue of the variable (can be NULL for empty).
Returns
1 on success, or an error code otherwise.
