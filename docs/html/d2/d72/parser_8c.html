Tokenize and parse a single command-line string into a list of commands.

This function performs tokenization of the input line and converts the resulting tokens into t_command objects which are appended to the provided commands list. The commands list must be created by the caller (for example via gen_list_create) and will be populated on success.

The function returns a status code defined in ms_status_codes.h:

  • MS_OK on success (commands appended to commands).
  • PARSER_ERR or other error codes on failure.
Parameters
lineNull-terminated input string to parse.
commandsOutput list where parsed t_command* objects will be appended.
Returns
MS_OK on success, or an error status otherwise.
