bool history_add ( char *  line)

Adds a command line to the shell history, with minimal validation.

The function first checks the given line:

  • Filters out empty or whitespace-only commands.
  • Avoids adding lines starting/ending with |.
  • Ensures that unclosed quotes are not stored.

If the command passes these checks, it is appended to the history using add_history(). Otherwise, nothing is added.

Parameters
lineA null-terminated string containing the command to be stored.
Returns
true if the command was successfully added to the history, false if it was rejected.
