Splits a raw command line into a list of tokens.

Parameters
lineA null-terminated string containing the input command line.
Returns
A pointer to a t_gen_list containing t_token* elements, or NULL if tokenization fails.

Each token corresponds to a command, argument, or operator, as defined in t_token_type.
