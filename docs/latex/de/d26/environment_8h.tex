{\footnotesize\ttfamily char \texorpdfstring{$\ast$}{*} env\+\_\+expand\+\_\+vars (\begin{DoxyParamCaption}\item[{\mbox{\hyperlink{environment_8h_a5ffba7f2f799b29cda5336b21a9990e6}{t\+\_\+environment}} \texorpdfstring{$\ast$}{*}}]{env,  }\item[{char \texorpdfstring{$\ast$}{*}}]{line }\end{DoxyParamCaption})}

Expands environment variables in a given string.

This function scans the input {\ttfamily line} for environment variable references prefixed with {\ttfamily \$} and replaces them with their corresponding values from the provided environment. The resulting string preserves other characters as-\/is and handles multiple variable expansions in the same line.

Example\+: If {\ttfamily line} is "{}\+Hello \$\+USER"{} and the environment has USER="{}\+Alice"{}, the function will return "{}\+Hello Alice"{}.


\begin{DoxyParams}{Parameters}
{\em env} & Environment instance. \\
\hline
{\em line} & Pointer to a null-\/terminated string that may contain environment variable references to expand.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
A newly allocated string with all recognized environment variables expanded. Returns NULL if {\ttfamily line} is NULL. The caller is responsible for freeing the returned string.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
This function handles only simple variable expansions (e.\+g., \$\+VAR) and does not process complex shell syntax like \$\{VAR\}. 
\end{DoxyNote}
