Parses a raw command line string into a structured list of commands.

This function is the high-\/level entry point for transforming user input into executable command objects. It performs the following steps\+:


\begin{DoxyEnumerate}
\item Tokenizes the input string using.
\item Converts the list of tokens into command objects.
\item Cleans up the intermediate token list.
\end{DoxyEnumerate}


\begin{DoxyParams}{Parameters}

\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
A list (t\+\_\+gen\+\_\+list) of t\+\_\+command\texorpdfstring{$\ast$}{*} objects representing the parsed commands, or NULL if parsing fails at any stage.
\end{DoxyReturn}
Example\+: Input\+: "{}ls -\/l \texorpdfstring{$\vert$}{|} grep txt $>$ out.\+txt"{} Output\+: A list of two t\+\_\+command structures\+:
