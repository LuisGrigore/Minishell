Tokenize and parse a single command-\/line string into a list of commands.

This function performs tokenization of the input {\ttfamily line} and converts the resulting tokens into {\ttfamily t\+\_\+command} objects which are appended to the provided {\ttfamily commands} list. The {\ttfamily commands} list must be created by the caller (for example via {\ttfamily gen\+\_\+list\+\_\+create}) and will be populated on success.

The function returns a status code defined in {\ttfamily \doxylink{ms__status__codes_8h}{ms\+\_\+status\+\_\+codes.\+h}}\+:
\begin{DoxyItemize}
\item MS\+\_\+\+OK on success (commands appended to {\ttfamily commands}).
\item PARSER\+\_\+\+ERR or other error codes on failure.
\end{DoxyItemize}
