{\footnotesize\ttfamily bool history\+\_\+add (\begin{DoxyParamCaption}\item[{char \texorpdfstring{$\ast$}{*}}]{line }\end{DoxyParamCaption})}

Adds a command line to the shell history, with minimal validation.

The function first checks the given line\+:
\begin{DoxyItemize}
\item Filters out empty or whitespace-\/only commands.
\item Avoids adding lines starting/ending with {\ttfamily \texorpdfstring{$\vert$}{|}}.
\item Ensures that unclosed quotes are not stored.
\end{DoxyItemize}

If the command passes these checks, it is appended to the history using {\ttfamily add\+\_\+history()}. Otherwise, nothing is added.

{\em line} & A null-\/terminated string containing the command to be stored.\\

\end{DoxyParams}
