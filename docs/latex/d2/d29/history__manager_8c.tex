{\footnotesize\ttfamily bool history\+\_\+add (\begin{DoxyParamCaption}\item[{char \texorpdfstring{$\ast$}{*}}]{line }\end{DoxyParamCaption})}



Adds a valid command line to the shell history. 

Validates and stores command lines in the shell history for later recall. Performs the following validations before storing\+:
\begin{DoxyItemize}
\item Rejects empty or whitespace-\/only commands
\item Rejects lines starting/ending with pipe operator
\item Rejects commands with unclosed quotes
\item Removes duplicate consecutive entries
\end{DoxyItemize}


\begin{DoxyParams}{Parameters}
{\em line} & The command line to store in history \\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
true if stored successfully, false if rejected
\end{DoxyReturn}
