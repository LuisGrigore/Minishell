Expands environment variables in a given string.

This function scans the input {\ttfamily line} for environment variable references prefixed with {\ttfamily \$} and replaces them with their corresponding values from the provided environment list {\ttfamily env}. The resulting string preserves other characters as-\/is and handles multiple variable expansions in the same line.

Example\+: If {\ttfamily line} is "{}\+Hello \$\+USER"{} and the environment has USER="{}\+Alice"{}, the function will return "{}\+Hello Alice"{}.
